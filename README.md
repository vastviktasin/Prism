# PRism

AI-powered GitHub Pull Request Intelligence Platform.

PRism transforms raw pull requests into:
- AI-generated summaries
- Risk analysis
- Complexity scoring
- Architecture-aware insights
- Developer-friendly review intelligence

---

## Features

- GitHub PR parsing
- AI PR summaries using Groq
- Risk score generation
- Complexity analysis
- Modern animated UI
- Responsive frontend
- Express.js backend
- Real-time PR insights

---

## Tech Stack

### Frontend
- React
- Vite
- CSS3

### Backend
- Node.js
- Express.js
- Octokit
- Groq SDK

## Installation

### Clone repository

```bash
git clone https://github.com/vastviktasin/Prism.git
```

### Install dependencies

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
nodemon server.js
```

---

## Environment Variables

Create `.env` inside backend:

```env
GITHUB_TOKEN=your_token
GROQ_API_KEY=your_key
```
